Medical Data Visualizer

Utilized python libraries' pandas, matplotlib, and seaborn to make calculations and visualizations from medical examination.

Project Description

The rows in the dataset represent patients and the columns represent information like body measurements, results from various blood tests, and lifestyle choices.

Dataset Shape: 70,000 rows, 14 columns
